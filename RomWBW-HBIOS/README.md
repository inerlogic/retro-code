# RomWBW / HBIOS

Tools here talk directly to RomWBW's HBIOS layer (disk parameter blocks, allocation vectors, etc.) rather than to any one machine's hardware. Anything filed here should run unmodified on **any** RomWBW-based board, regardless of processor — currently that's the SC131 (Z180) and, when back in use, the RetroBrew-SBC and RC2014 (both Z80).
